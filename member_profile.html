Member Profile

Elements

The member profile is not only the primary access point for members to view other member's personal information, it also provides primary administration access to the profile holder.

  • profile picture
  • actions button
  • ribbons
  • social links
  • profile holder name and title
  • new message indicator - right justified
  • tabs:
    • my profile
      • my speaker's profile
      • my mentor profile
    • my connections
      • my contacts
      • my networks
      • my communities
    • my Account
      • inbox
      • privacy settings
      • email preferences
      • RSS feeds
      • subscriptions
      • discussion signature
      • admin
      • change password
  • bio
  • education history
  • job history
  • demographics
