Data

Data tables can be represented in two ways - as a table or as a structured bootstrap grid (a.k.a aggregate table data).

Data Table Elements

  • data tables are for tabular data
  • count / sort / filter / actions above the header
  • header bar
    • column headings
    • action column (as is needed to enable functionality)
    • shows actions based on relationship state, access and permissions
  • data in rows

Data Grid Elements

Data grids are responsive and typically have aggregate or summary data in a given row. Unlike the data tables, data grids designed to present a context for more complex information.

  • bootstrap grid filter row
  • count - is always on the left in this format: Showing X to X of X
  • sort / filter / actions are to the right
  • bootstrap header bar
    • column headings
    • action column (as is needed to enable functionality)
    • shows actions based on relationship state, access and permissions
  • data elements aggregated into logical groupings in rows and columns
