Tabs

Description

Add quick, dynamic tab functionality to transition through panes of local content, even via dropdown menus.

Elements

  • bootstrap tabs
  • on-state for active tab
  • can have labels for summary information
  • can have dropdowns for sub-navigation
  • content and functionality inside tab must persist tab environment for aggregated pages (community landing and profile)

Examples

Normal States

Colors provide visual cues for users, if absensce of a priority/severity do not display.
