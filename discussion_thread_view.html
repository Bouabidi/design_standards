Discussion Thread View

Elements

  • "Back to Discussions" button - returns to thread list
  • topic title
  • original topic author
    • profile picture
    • name
    • timestamp
    • summary snippet from message body
  • threaded topic responses
  • response blocks - first at the top and decending
    • number and subject line (if best answer, icon appears)
    • recommends - right justified in title
    • profile picture
    • name
    • actions button - related to author profile
    • timestamp
    • email body
    • show original message block - expands and collapses
    • multi-action dropdown button - based on permissions and configuration
