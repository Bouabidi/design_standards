Filter & Sort Options

Filter and sort options appear above the header of most all list pages or widgets. The combination or sort and fitler options depend entirely on the given use cases of a particular list. Some lists only have a sort option dropdown while others may have multiple filter options, a search input and sort options... again, it depends on the use cases.

Rules & Guidelines

  • reside at the top of lists
  • options dropdown represent sortable columns on the page
  • if data does not exist, sort option should not show in the list
  • on refresh, the chosen sort options should be visible in the dropdown to inform user of list sort
