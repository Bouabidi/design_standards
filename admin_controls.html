Admin Controls

Description

Admin buttons appear as needed based on user's access permissions

Rules & Guidelines

  • multi-action dropdown (community admin)
  • single-action buttons for control admin
  • interative content actions with block-level "create/add" and inline "edit" and "delete"
  • persistent admin for site administation
