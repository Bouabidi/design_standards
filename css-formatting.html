CSS Formatting and Targeting

There are a number of ways to format and target CSS. Higher Logic's CSS is formatted in an indented, multi-line format. There are a number of ways to target CSS and it's up to each developer to determine which works best in any given situation. The goals here are to write CSS that does what it is intended to do accross all supported browsers and it does so without unintentially impacting other content.


Formatting

There are many ways to format CSS but we're using the multi-line format with indenting option shown below.

Multi-line indented format

For cross browser compatibility, make sure all hex colors are fully defined - no abbreviations. Use #333333 instead of #333.

Multi-line indented format


Targeting

Understanding how to target your CSS so that it applies only where intended is critical for proper UI display. Avoid broadly defined CSS unless you're certain it is the proper solution. The links below explain CSS targeting and how specificty is calculated.
