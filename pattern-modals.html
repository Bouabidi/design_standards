Modals

There are two types of modals: action modals and information modals. Action modals present form inputs for content creation and configuration and require a save or cancel action. Information modals present explanations, instructions or help text and no other action except closing the modal.

Types of Modals

  • Action modal - presents form elements and requires a Save or Cancel action.
  • Information modal - presents information, explanation or help text and requires a Close action.
  • Confirmation Prompt modal - prompts user for confirmation before executing the action.

Modal Sizes

  • Default size (bootstrap native), 650px wide.
  • Wide size, 800px wide. Class "modal-wide" can be added to the modal-content div.

Guidelines

  • When a modal is open, there is no user access to any action or information behind it. The user must use the action buttons in the modal to close it.
  • All modals should have the "X" close action in the upper right.
    • For action modals, the "X" has the exact same behavior as the cancel action.
    • For information modals, the "X" has the exact same action as the close action.

Structure

  • Background layer.
  • Header - displays the title and the "X" cancel/close action.
  • Body - displays the modal content.
  • Footer - displays the action buttons.
