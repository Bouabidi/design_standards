Search Results

Search results is a unique pattern in our system. All content types have common components and each content type may have additional components needed to display content-specific information.

Types of Content

  • Announcements
  • Blogs
  • Community
  • Discussions
  • Discussion Thread
  • Discussion Posts
  • Event
  • Event Session
  • Glossary Terms
  • Libraries
  • Library Entries
  • Library Files
  • Page Content
  • Volunteer Opportunity

Structure

  • Title
  • Type Label
  • Snippet Summary
  • Tags
  • Attachement
  • Dates
  • Focused Search
  • Author/Owner
    • Photo
  • Publication Date
  • Content Location
