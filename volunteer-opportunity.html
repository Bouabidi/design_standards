Volunteer Opportunity

Elements

  • title
  • multi-action dropdown button
  • date and time information
  • event details
    • description
    • location
    • event icon/graphic
    • action button(s) - dependent on event setup/configuration and registration state
      • RSVP
      • Register
      • Cancel RSVP
      • Delete/Cancel Registration
      • Register Others
    • "Download to Calendar" button
    • event contact information
