Library Entry

Elements

Library entries are complex and based on parameter configurations.

  • "Back to Library" button - returns to library entry list
  • entry title
  • recommend block in title - right justified
  • author block
    • profile picture
    • name
    • summary snippet from message body
  • timestamp
  • statistics
  • attachments list:
    • type icon
    • file name
    • document size label
    • revision history
    • download option
  • tags and keywords (see tags for details)
  • comments (see comments for details)
  • related links (see related links for details)
