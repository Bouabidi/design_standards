CSS Solution Structure

Our product's CSS structure is found in the HigherLogic/CSS directory. It's broken into multiple directories and files matching our product modules and functionality to make management as logical as possible.

As the need arises, add forward facing CSS to the appropriate CSS file in the BootstrapUX directory and admin facing CSS to the appropriate CSS file in the CCAdmin directory. See rules for adding CS for more details.

If the need arises to create a new directory or files, please consult with either Rick Schott or Patrick Bragg prior to adding to the solution.

  • BootstrapUX directory houses all the CSS for rendering the product for our clients.
  • BootstrapUX/boostrap-combined.min.css is our Bootstrap 3.0.1 master CSS file and should never be modified by anyone.
  • CCAdmin contains all the CSS for the CCAdmin site.
  • Desktop contains all the CSS for non-Bootstrap product - nothing should be added to this directory.
  • MobilePhone contains all the CSS for JQuery Mobile verion, no longer used - nothing should be added to this directory.
  • Shared contains CSS that is shared between CCAdmin and BootstrapUX. Stylistically, our product and admin sitees are quite different and only in specific cases should any CSS be added to this directory.
