Responsive Grid

Bootstrap's responsive grid handles all responsive needs automatically.

Overview

Higher Logic's implementation of the Bootstrap media queries are structured with max-width (desktop first). We don't use min-width (mobile first) in our base because the entire application was built as a desktop application. Adding CSS to the media queries directly should never be done unless Patrick Bragg or Rick Schott are in the loop. Override CSS that includes media queries is acceptable if the targeting is precice and verified.
