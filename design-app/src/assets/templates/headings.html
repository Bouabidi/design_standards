Headings - H1 thru H4

Headings are an important part of the page structure. Not only do they set the information priority and the hierachy of a page's content but they also provide structure for page readers.

When to use headings

  • the masterpage automatically inserts the H1 on every page, unless it is configured not to appear by the client, so you'll never add an H1 to any control.
  • if your coding a ContentUserControl, then use an H2 as the headings/title.
  • H3 and H4 should be used only if there is a clear heirachy or structure to present.
