Help and Assistive Text

Description

The tooltip plugin generates content and markup on demand, and by default places tooltips after their trigger element.

Rules & Guidelines

  • Modal - when more space is needed to convey the point.
  • Popover via an icon - when less space is needed to convey the point.
  • Title attribute added to links - inside all anchors to help screen readers.
  • Tooltip - when very concise tips can be conveyed in a small amount of space.
  • Help-block
  • Help-block with "NOTE:"

Help Block

Persistent help block added below the form element.



Example of instructional text to be put here if needed.

HTML

Copy <div class="form-horizontal">
<div class="form-group">
<label class="col-sm-1 control-label" for="inputPassword3">Choose</label>
<div class="col-sm-6">
<select class="form-control">
<option>1</option>
</select>
<div class="help-block">Example of instructional text to be put here if needed.</div>
</div>
</div>
</div>



Help Block with Alert

Note:  Instructional text with the NOTE emphasis can be used to give it more priority. This should be used only when emphasis is truly needed.

HTML

Copy <div class="form-horizontal">
<div class="form-group">
<label class="col-sm-1 control-label" for="inputPassword3">Choose</label>
<div class="col-sm-6">
<select class="form-control">
<option>1</option>
</select>
<div class="help-block"><span class="alert-info note-alert">Note:</span>&nbsp; Instructional text with the NOTE emphasis can be used to give it more priority. This should be used only when emphasis is truly needed.</div>
</div>
</div>
</div>
