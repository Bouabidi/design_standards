Site Page Structure

Higher Logic Masterpage

Higher Logic's implementation of the Bootstrap 3 scaffolding (grid) accommodates proper rendering of our modules and controls. Understanding this structure is essential to those applying custom designs to their microsites. Inspect the HTML using Firebug or other HTML plugins or right-clicking - View Source to explore the scaffolding as well.

 




Site Content Frame Size

Narrow - 984px wide

This is the same default content frame width as pre-Bootstrap sites.

  • go to Site Setup / Layout in the site admin and select Narrow from the dropdown.

 



Wide - 1200px wide

Bootstrap 3 content frame default width.

  • go to Site Setup / Layout in the site admin and select Wide from the dropdown.

 



Full Frame - 100% of the browser window

This configuration setting accommodates site designs with site background colors and/or images stretching the entire width of the browser frame. See "Full-frame site design".

  • go to Site Setup / Layout in the site admin and select Full from the dropdown.

Site Logo

The site logo is uploaded on the Site Setup / Images page. The size of the logo is completely subjective. A ballpark size may be anywhere from 250px width by 100px height to 300px wide by 80px height. Below are considerations for proper logo treatment.

  • The logo creates physical space within the header so the height and width of the logo has a direct impact on the height of the header.
  • Smaller Logo
  • Larger Logo
  • For optimal responsive display, it's best to keep logo width under 320px.
  • Logo images must be separate from header background images to ensure proper responsiveness and display on all devices. We no longer support integrated header background and logo images in a single graphic.
    • Upload the site logo by going to the Site Setup / Images and choosing Logo.
    • If your design requires a header background image, upload it on the Site Setup / Images page using Header Background choice.
    • NOTE: A single header background image should be sized to fit the Site Frame setting - Narrow 894px, Wide 1200px and Full (custom size).

Slidedeck Images

Images loaded into the Slidedeck control should match the setting for the site frame to display properly.

  • If the site frame is Narrow, images should be no greater than 984px wide
  • If the site frame is Wide, images should be no greater thatn 1200px wide

  • NOTE: use UseResponsive="true" to make Slidedeck images responsive to smaller devices

Integrating Bootstrap 3.0.1 Themes and Templates

Templates

There are two ways to add a Bootstrap 3.0.1 theme: cut and paste the theme's CSS into a new or existing theme; or upload a CSS file into a new or existing. It's important to note the difference between a Theme and a Template because often both words are used to mean the same thing when in fact, they are quite different.

  • A Template, purchased and/or downloaded from sites like Wrapbootstrap.com and others typically contains structure(XHTML), interaction scripts(Javascript) and design(CSS and graphics). Most are created for use in open CMS systems like Word Press.com or Joomla.org, systems with very little to no inherent structure - they are a "blank slate" - and are designed to accept most any template design.
  • A typical Bootstrap template
  • Importing a Template into a Higher Logic theme requires specialized knowledge of HTML and CSS and a firm understanding of both the Bootstrap 3.0.1 grid and Higher Logic's implementation of the grid (see Site Structure). A template's custom structural elements must be translated into Higher Logic's custom structure as well as any other custom interaction scripts or widgets associated with before a template is functionally integrated.
  • A typical Bootstrap template

Themes

  • A Theme, as Higher Logic refers to them, is comprised of CSS defining a site's look and feel. Themes that define Bootstrap elements and components work very well with Higher Logic's content management system (CMS) since it contains a custom theme structure to accommodate our social tools and functionality. Bootswatch.com is a recommended site with free themes.
  • A typical Bootstrap theme
  • A Theme holds the color and spatial definitions required to meet design specifications. It can be a combination of CSS.
  • A typical Bootstrap theme
