Breadcrumb

Description

Indicate the current page's location within a navigational hierarchy.

Rules & Guidelines

  • link home
  • label of current location within site

Examples

Default breadcrumbs

Separators are automatically added.

  1. Home
  2. Library
  3. Data
<ol class="breadcrumb">

   <li><a href="#">Home</a></li>

   <li><a href="#">Library</a></li>

   <li class="active">Data</li>

</ol>
